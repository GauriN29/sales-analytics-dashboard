# Sales Analytics Dashboard

An interactive Sales Analytics Dashboard built using Python, Pandas, Streamlit, and Matplotlib.

## Features

* KPI Cards (Sales, Profit, Orders, Customers)
* Interactive Filters (Year, Region, Category)
* Yearly Sales Trend
* Monthly Sales Trend
* Sales by Category Pie Chart
* Top 10 Sub-Categories Analysis
* Business Insights
* Download Filtered Data

## Technologies Used

* Python
* Pandas
* Streamlit
* Matplotlib

## How to Run

1. Install dependencies

```bash
pip install -r requirements.txt
```

2. Run the application

```bash
streamlit run app.py
```
